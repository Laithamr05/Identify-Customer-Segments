# Identify Customer Segments

This repository contains the completed Udacity Arvato customer segmentation project deliverables:

- `Identify_Customer_Segments.ipynb`: the finished notebook
- `Identify_Customer_Segments.html`: the exported HTML version of the notebook

## Project Summary

The notebook applies an unsupervised learning workflow to demographic data in order to:

- clean and preprocess the Arvato population and customer datasets
- engineer mixed and categorical features
- scale features and reduce dimensionality with PCA
- cluster the population with k-means
- compare customer cluster membership against the general population

## Notes

- The original Arvato datasets are not included in this repository.
- To view the analysis without Jupyter, open `Identify_Customer_Segments.html`.
